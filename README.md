# 🏍️ Ramp Rider

A simple 2D motorcycle scroller. Ride at a constant speed over ramps, jump obstacles
and chasms, land on your wheels, and grab power-ups to clear the big jumps — all the
way to the finish line.

**Play it:** open `index.html` in any modern browser. No build step, no dependencies.

## Controls

| Action | Keyboard | Touch |
| --- | --- | --- |
| Jump | Space / ↑ / W | **JUMP** button |
| Spin / lean back | ← / A | **◀** button |
| Spin / lean forward | → / D | **▶** button |
| Restart | Enter / R | tap the screen |

- **Land on your wheels.** A landing is safe as long as your center of gravity stays
  between the wheels — tilt too far and you wipe out.
- **⚡ Boost** power-ups give you speed (distance) for wide chasms.
- **⏫ Double-jump** power-ups give you a mid-air jump (height) for the tall towers.

## Built with

Plain HTML5 Canvas + vanilla JavaScript — a hand-rolled physics engine, a chunk-composed
level, and a dusk-silhouette art style.
